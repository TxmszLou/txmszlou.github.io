Ti84/83 Game 2048

Posted on April 1, 2014
Tags: Ti83/84, Game Development

0x00 What is 2048?

2048 is a html5 game created by Gabriele Cirulli. Based on 1024 by Veewo Studio and conceptually similar to Threes by Asher Vollmer.

0x01 What is 2048-ti?

2048-ti is a implementation for the html5 game 2048 on Ti83/84 calc. It simplified the animation while main functions are kept.

{% limg 2048-ti.gif {“alt”:“2048-Ti”}%}

Github

0x02 Installiaion

  1. Using Ti-connect or other connecting software copy file TWOZERO.8Xp to your Ti83/84 calc.
  2. Launch the program using built in function Asm(). //Press [2nd]+[0] go to the catalog, function Asm() is on the first page

0x03 For developers

  1. Get the source code in your calc by transferring P2048SRC.8Xp to your calc.
  2. Having look at the source code online -> P2048SRC.txt.

0x04 BUG LIST:

  1. Automatically merge block within one step. (e.g. 4,4,4,4 in a row, after pressing [<-/->], you might get a single 16 rather than two 8s).
