Posted on February 5, 2014
Tags: Ti83/84, Tutorial

Leafiness0’s Particular Explosion Tutorial 翻译:ThomazL 转载请著名出处:「ti83/84」 粒子爆炸教程

梗概: 再本教程中，你会学会如何用数组中储存的粒子来制造爆炸效果

你们可能会想 —— 我怎么才能做出像Stick Ninja, Tag, Graviter等游戏中的碉炸天粒子爆炸效果？ 粒子爆炸效果虽然看起来十分棘手，但事实上是可以十分容易地创建的…

Repeat getkey(15)

End

把代码放在一起:

For(r1,0,79)
r1*8+L1->r6
12288->{r6}{r}
12288->{r6+2}{r}rand^300-150->{r6+4}{r}
rand^300-325->{r6+6}{r}
End
Repeat getKey(15)
For(r1,0,79)
r1*8+L1->r6
{r6}{r}+{r6+4}{r}->{r6}{r}
{r6+6}{r}+4->{r6+6}{r}
{r6+2}{r}+{r6+6}{r}->{r6+2}{r}
Pxl-On({r6}{r}/256,{r6+2}{r}/256)
End
DispGraph
ClrDraw
End
