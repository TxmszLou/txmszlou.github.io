WHAT

A ti-calc(83/84) music game written in AXE language.

HOW:

(1)Send the main program {TIBEAT.8xp} (no shells) and the music chart {MYMAP.8xv} to the calculator.

(2)lanuch the game by using asm() from the home screen.

(3)Click keys with the blocks. Four keys(D,E,F,G) represent four blocks form left to reight.

Playing music from the calculator is not supported yet.

FOR DEVELOPERS:

{ASMTEST.8xp}– a axe program to create music flies in appvar.

{OO.8xp}– a axe program to test the music chart and see if there is a problem.

{MYMAP.8xv}– the appvar careated from {ASMTEST.8xp}.
