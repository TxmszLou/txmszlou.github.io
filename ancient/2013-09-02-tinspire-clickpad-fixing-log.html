TiNspire ClickPad Rescue Log

Posted on September 2, 2013
Tags:

It started when I bought a really realy really cheap Ti nspire on Taobao…

Though it’s really cheap (around a quarter of the original), and it’s not a prototype…

I still found problems as I tested it…

Can get it upgrade on to OS 3.1.0…

Details

  1. Upgrade error using TInspireconnector.

All right with the transferring, fail to install, prompt:

This OS is not suitable for the machine...
  1. Same error prompt after cleaning the data and OS.

Problem

boot2 version too low

Solution

  1. Upgrade to OS 1.6.0 using TInspireconnector.

  2. Upgrade boot2 to version 1.4.x.

  3. Upgrade to OS 3.1.0.

  4. Upgrade boot2 to version 3.1.x.
